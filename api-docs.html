API 文档

使用 LumeFlow API 将 AI 视频生成能力集成到您的应用中。

基础 URL

https://api.lumeflow.com/v1

认证

所有 API 请求需要通过 API Key 进行认证：

Authorization: Bearer YOUR_API_KEY

您可以在个人中心的 API 设置页面获取 API Key。

创建视频

POST /videos/generate

创建一个新的视频生成任务。

请求参数：

参数 类型 必填 描述
prompt string 是 视频描述文本
mode string 否 模式：text 或 image
duration number 否 视频时长（秒），默认 3
style string 否 视频风格

示例请求：

curl -X POST https://api.lumeflow.com/v1/videos/generate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "prompt": "一只可爱的猫咪在草地上奔跑", "duration": 5, "style": "realistic" }'

查询状态

GET /videos/{video_id}/status

查询视频生成状态。

响应示例：

{ "id": "vid_abc123", "status": "completed", "progress": 100, "video_url": "https://cdn.lumeflow.com/videos/abc123.mp4", "created_at": "2026-03-02T10:00:00Z" }

状态说明：

  • pending - 等待处理
  • processing - 生成中
  • completed - 已完成
  • failed - 生成失败

视频列表

GET /videos

获取当前用户的所有视频。

查询参数：

  • limit - 返回数量，默认 20
  • offset - 偏移量，默认 0

Webhooks

配置 Webhook 以接收视频生成完成的通知：

事件类型：

  • video.completed - 视频生成完成
  • video.failed - 视频生成失败

Webhook 载荷：

{ "event": "video.completed", "video_id": "vid_abc123", "video_url": "https://cdn.lumeflow.com/...", "timestamp": "2026-03-02T10:30:00Z" }

速率限制

免费用户：60 请求/分钟
付费用户：300 请求/分钟
